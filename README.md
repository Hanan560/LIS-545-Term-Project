# LIS-545-Term-Project (Data Curation)
Overview - This repository contains a curated documentation for the following dataset:

Cribbs, H. L., & Gardner, G. J. (2022).
Data for “To Pre-Filter, or Not to Pre-Filter: That Is the Query: A Multi-Campus Big Data Study”. Figshare. https://doi.org/10.6084/m9.figshare.19071578.v1

Here is the associated publication to the dataset: 
Cribbs, H. L., & Gardner, G. J. (2022). To pre-filter, or not to pre-filter, that is the query: A multi-campus big data study. Journal of Librarianship and Information Science, 55(4), 1119-1142. https://doi-org.offcampus.lib.washington.edu/10.1177/09610006221124609 (Original work published 2023)

List of files:
1. 2017-2019_RAW.csv: Raw search interaction logs exported from Ex Libris Primo Analytics (OBIEE) for all 24 CalState libraries for calendar years 2017 - 2019.
2. clean_data.csv: Cleaned dataset prepared for statistical analysis.
3. CCIHE2018-PublicDataFile.xlsx: Carnegie Classification data from the Indiana Univeristy Center used to contextualize institutional characteristics.
4. IA screenshots CSU Libraries search config.pdf: Archived screenshots of CSU library discovery interfaces captured from the Internet Archive's Wayback Machine.
5. NonparametricStatisticalTestsFromSPSS.zip: Contains 23 SPSS output files (.spv format) documenting statistical test results and analytical outputs from the study (useful for reproducibility purposes).
6. datacite.xml: Metadata that was generated using Microsoft Copilot and then manually reviewed and edited for schema compliance and accuracy.
